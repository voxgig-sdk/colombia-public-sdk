# colombia-public-mcp

MCP server exposing the ColombiaPublic SDK as tools, built on the
[official Go MCP SDK](https://github.com/modelcontextprotocol/go-sdk)
and the sibling Go SDK at `../go`.

## Tools

| Tool | Args | Returns |
|------|------|---------|
| `colombia-public_list` | `entity`, optional `query` map | First page of records as JSON |
| `colombia-public_load` | `entity`, `query` (e.g. `{id:N}`) | Single record as JSON |

JSON schemas for both tools are emitted by the SDK from the `Args`
struct's `json` / `jsonschema` tags — no schema is hand-written.

## Entities

airport | category_natural_area | constitution_article | country | department | holiday | invasive_specie | map | native_community | natural_area | president | radio | region | touristic_attraction | typical_dish

## Build

```sh
go build -o colombia-public-mcp ./...
```

## Run

```sh
# stdio transport — for spawnable agent installs (default).
./colombia-public-mcp

# streamable HTTP transport — share one running server between agents.
./colombia-public-mcp -transport http -addr :8080
```

## Install for Claude Code

```sh
claude mcp add --scope user colombia-public \
  -- /absolute/path/to/colombia-public-mcp -transport stdio
```

After install, restart Claude Code; the `colombia-public_list` and
`colombia-public_load` tools become available in new sessions.

## Smoke test via HTTP

```sh
./colombia-public-mcp -transport http -addr :18080 &

# initialize, grab the session id
curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -D headers \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-06-18","capabilities":{},"clientInfo":{"name":"smoke","version":"0"}}}'

SESSION=$(awk '/Mcp-Session-Id/ {print $2}' headers | tr -d '\r')

curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -H "Mcp-Session-Id: $SESSION" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"colombia-public_load","arguments":{"entity":"airport","query":{"id":1}}}}'
```
