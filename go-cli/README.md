# colombia-public-cli

AQL-driven CLI and REPL for the ColombiaPublic SDK. Positional arguments are
joined into a single AQL expression and evaluated; with no arguments,
falls into an interactive REPL.

Built on `github.com/aql-lang/aql/eng/go` and the sibling Go SDK at `../go`.

## Build

```sh
go build -o colombia-public-cli ./...
```

## Run

```sh
# One-shot: arguments form a single AQL expression
./colombia-public-cli list airport
./colombia-public-cli load 1 airport
./colombia-public-cli load '{id:1}' airport

# REPL
./colombia-public-cli
```

## Words

| Word     | Signatures                                   | Description                |
|----------|----------------------------------------------|----------------------------|
| `list`   | `[entity]` · `[query entity]`                | List records               |
| `load`   | `[entity]` · `[query entity]`                | Load a single record       |

`query` is either a Map (`{id:1}`) or a Scalar (`1`, treated as `{id:1}`).
`entity` is one of the SDK's entity names (auto-quoted as an atom).

## Entities

airport category_natural_area constitution_article country department holiday invasive_specie map native_community natural_area president radio region touristic_attraction typical_dish

## REPL commands

- `:quit` / `:q` / `:exit` — exit the REPL
- `:help` / `:h` / `:?`     — show help
